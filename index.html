About Me

Hey, I’m Aaron — a self-taught developer from Boston with a passion for building clean, smart, and purposeful applications. I’m currently focused on full-stack JavaScript, but I’ve got a solid Python foundation and a deep love for computer science fundamentals like algorithms, databases, and system design.


I didn’t take the traditional path into tech — instead, I built my own curriculum using Codecademy, Udemy, and a stack of books, fueled by caffeine, curiosity, and cold plunges (seriously, face-dipping in ice water beats coffee any day). I’m not learning to land a job — I’m learning to build my own apps, like Poolio, a savings and investing platform for groups. Every line of code I write moves me closer to that vision.


When I’m not knee-deep in JavaScript, Python, or PostgreSQL, I’m probably optimizing my study setup with noise-canceling headphones and fasting to stay laser-focused. I believe programming is a superpower, and I’m here to wield it with precision, creativity, and just a bit of fun.

Reddit Minimal

Reddit Minimal is a clean, streamlined Reddit client built with React and Redux that focuses on delivering a distraction-free browsing experience. The application fetches data from Reddit's JSON API to display posts with a minimalist interface, featuring smooth animations, responsive design, and intuitive navigation. Users can browse different subreddits, view post details, and enjoy a clutter-free alternative to the traditional Reddit interface that prioritizes readability and performance.
